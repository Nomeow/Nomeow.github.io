C#笔记——面向对象

发表于 2018-05-29 | 分类于 C#

类：

程序的数据和功能被组织为逻辑上相关的数据项和函数的封装集合，被称为类，类是一个能储存并执行代码的数据结构。他包含 数据成员 和 函数成员。

数据成员：字段，常量。表示类或类的实例相关的数据（有什么）
函数成员：方法，属性，运算符，索引器，事件。表示类的实例能执行的操作（能干什么）

类成员

字段和方法是最重要的类成员类型。字段是数据成员，方法是函数成员。

字段

字段是隶属于类的变量

如果没有初始化语句，字段的值会被编译器设为默认值，默认值为字段的类型决定。值类型的默认值为1，bool类型的默认值为法拉瑟，引用类型的默认值为null。

属性

用于描述类的特征。属性通常带有get和set访问器，get用来获取属性的值，set用来设置属性的值。

如想让属性变为只读，则可以去掉set访问器，只保留get访问器。

字段是真正存储数值的变量，属性只是对外公开的窗口。

方法

构造函数与析构函数

构造函数在创建类的实例时被调用，他：
（1）名称必须与类名相同
（2）没有返回值
（3）默认构造函数无参数，但也可以定义参数

析构函数以“~”开头，无返回值，后紧跟类名，无参数。他在资源被销毁是被调用。

创建类的实例

类是引用类型，所以要为数据引用和实际数据都申请内存。此时要用到 new 运算符
new 运算符为指定类型的实例分配并初始化内存，依据类型的不同从栈或堆里分配。

结构

结构与类比较相似，同样包含字段属性方法。但是：
（1）结构只能声明带参数的构造函数。
（2）结构不能进行继承和派生，但可以实现接口。
（3）结构在实例化时可以忽略new关键字。
