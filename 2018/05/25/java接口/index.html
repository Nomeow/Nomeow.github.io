Java笔记——接口

发表于 2018-05-25 | 分类于 JAVA

接口的概述

1.接口中只能定义常量和抽象方法。
2.接口中的方法默认为public，abstract。

接口的实现

声明

在类的声明中用implements关键字来表示在类体中可以使用接口中的常量，而且必须实现（重写）接口中的所有方法。若使用多个接口则需以逗号隔开接口名。

回调

其他

1.接口可以作为引用类型来使用
2.任何接口类型的实例都存储在该接口类型的变量中
3.接口中的方法默认为public，所以在重写方法时应限定为public
