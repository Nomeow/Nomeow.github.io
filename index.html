PAT乙级练习——1007 母牛的故事

发表于 2018-10-31 | 分类于 PAT

题目描述
有一头母牛，它每年年初生一头小母牛。每头小母牛从第四个年头开始，每年年初也生一头小母牛。请编程实现在第n年的时候，共有多少头母牛？

输入描述:
输入数据由多个测试实例组成，每个测试实例占一行，包括一个整数n(0<n<55)，n的含义如题目中描述。

输出描述:
对于每个测试实例，输出在第n年的时候母牛的数量。
每个输出占一行。

输入例子:
2
4
5

输出例子:
2
4
6

注意：
每一年牛的数量等于去年的牛的数量，加上今年能生小牛的牛的数量

PAT乙级练习——1041 说反话

题目描述
给定一句英语，要求你编写程序，将句中所有单词的顺序颠倒输出。

输入描述:
测试输入包含一个测试用例，在一行内给出总长度不超过80的字符串。字符串由若干单词和若干空格组成，其中单词是由英文字母（大小写有区分）组成的字符串，单词之间用1个空格分开，输入保证句子末尾没有多余的空格。

输出例子:
Come I Here World Hello

（1）Split()分割字符串。
（2）Reverse()反转数组中的元素的序列。
（3）Trim()去掉字符串前后空格。

PAT乙级练习——1033 害死人不偿命的(3n+1)猜想

发表于 2018-10-29 | 分类于 PAT

卡拉兹(Callatz)猜想：

对任何一个自然数n，如果它是偶数，那么把它砍掉一半；如果它是奇数，那么把(3n+1)砍掉一半。这样一直反复砍下去，最后一定在某一步

得到n=1。卡拉兹在1950年的世界数学家大会上公布了这个猜想，传说当时耶鲁大学师生齐动员，拼命想证明这个貌似很傻很天真的命题，

结果闹得学生们无心学业，一心只证(3n+1)，以至于有人说这是一个阴谋，卡拉兹是在蓄意延缓美国数学界教学与科研的进展……

我们今天的题目不是证明卡拉兹猜想，而是对给定的任一不超过1000的正整数n，简单地数一下，需要多少步（砍几下）才能得到n=1？
输入描述:
每个测试输入包含1个测试用例，即给出自然数n的值。

输出描述:
输出从n计算到1需要的步数。
示例1
输入
3
输出
5

对海高的碎碎念

发表于 2018-06-24 | 分类于 杂谈

…
寻梦？撑一支长篙，

　　向青草更青处漫溯；

　　满载一船星辉，

　　在星辉斑斓里放歌。

　　但我不能放歌，

　　悄悄是别离的笙箫；

　　夏虫也为我沉默，

　　沉默是今晚的康桥！

　　悄悄的我走了，

　　正如我悄悄的来；

　　我挥一挥衣袖，

不带走一片云彩。

这首新月诗派最具代表性的作品创作于1928年，九十年后的2018年，我坐在这里为已挥别两年的海高写下这篇文字。

诗人游学剑桥时曾满怀深情地说：“我的眼是康桥叫我睁的，我的求知欲是康桥给我拨动的，我的自我意识是康桥给我胚胎的。”而我已可以说：“我的眼是母亲教我睁的，我的求知欲是海高给我拨动的，我的自我意识是老师给我胚胎的”
喜欢这首《再别康桥》已经很久了，读到这首诗的时候就更久了，久到还是刚上高中的时候，久到才上过高中第五节语文课的时候，久到大家还读不出来我的名字的时候。在学过《再别康桥》的那几天，在阳光漫射去的晨路上，在路灯晕染过的归途中，边走边在心中默诵这首诗。悄悄地我走了，不带走一片云彩，我来海高三年，未曾在学校里留下一点痕迹，未能在某个展板的角落里展示自己未曾存在过的骄傲战绩，也许在海高的下一个三十年，下下个三十年。只要提到“谢旭”二字，人们想起的绝不会是那个拗口又难认的“珩”字。只得借济慈的墓志铭：“我把名字写在水上”来安慰自己。没能在母校添上自己的一笔，但在梦开始的地方的各种感动，叛逆，辛苦，和惊喜还有每一位师长同学，注定在我心中留下了美好的回忆。三年修行结束，正像是许嵩在《如约而至》里的那句“人生看似偶然却又必经的告别”在高一的入学典礼上没有人会考虑自己三年后的样子，即便有也只是一串理想大学的名字，而三年后的毕业典礼上乃至毕业后的下一次毕业却必定有人在努力的回忆。

在大学校园里，语文课对于工科生而言早已被列入通识选修，就是大家心照不宣最不重要的那一类。但我还会时常想起高中语文课上《雷雨》里鲁大海被周朴园开除时周冲那天真的一句“爸爸，这是不公平的。”；还会时常想起一向很少出错的刘亦蒙在把《孔雀东南飞》中的刘兰芝读成刘志兰时的哄堂大笑；还会时常想起大家在思考如何工整的对上“作子修座子”时的苦思冥想；还会时常想起芳芳在黑板上默写古诗出错时自罚的那首《橄榄树》；还会时常想起在高三苦海中徜徉时每周最期待的《一轮明月》和《国宝档案》；还会时常想起高中最后一节语文课上芳芳送给我们的那首《滚滚红尘》,起初不经意的你,和少年不经事的我…分易分，聚难聚，爱与恨的千古愁…时常想起过去我们坐在教室里，熟悉的面孔，相嘻无故爆发出欢喜的笑声，装模做样听那无聊的魔鬼数学，背那自己都不知道有多诡异蹩脚发音的单词，时常想起往昔真切发生的点点滴滴，现在只能出现在记忆里。

在海高，我们奋斗三年，在海高，我们嘻笑三年，在海高，我们离别各自走向天涯海角。我们都年轻，一如既往地按着人人一致而又各具精彩的渠道通向自信满满的未来，一切都还在，高中母校里的那份执着和疯狂压在心灵深处，谁也偷不走。

沙扬娜拉。

C#笔记——类

发表于 2018-05-29 | 分类于 C#

类：
程序的数据和功能被组织为逻辑上相关的数据项和函数的封装集合，被称为类，类是一个能储存并执行代码的数据结构。他包含 数据成员 和 函数成员。

数据成员：字段，常量。表示类或类的实例相关的数据（有什么）
函数成员：方法，属性，运算符，索引器，事件。表示类的实例能执行的操作（能干什么）

类的声明

关键字class

类成员

字段和方法是最重要的类成员类型。字段是数据成员，方法是函数成员。

字段

字段是隶属于类的变量

如果没有初始化语句，字段的值会被编译器设为默认值，默认值为字段的类型决定。值类型的默认值为1，bool类型的默认值为法拉瑟，引用类型的默认值为null。

Java笔记——抽象类

发表于 2018-05-28 | 分类于 JAVA

关键字：abstract
作用：限制所有该抽象类的子类都有该抽象类的操作。

注意：
（1）抽象类不能创建对象，必须派生出子类由子类创建对象。
（2）抽象方法只允许声明不允许实现，即没有方法体。
（3）抽象类的子类必须重写父类中的所有抽象方法。
（4）抽象类中可以没有抽象方法，抽象方法必须放在抽象类中。

Java笔记——Object类

发表于 2018-05-28 | 分类于 JAVA

Object类： java中所有类的父类。

可以使用Object类型的变量引用任何类型的对象，但若想对其中的值进行操作还需要搞清对象的原始类型，并进行相应的类型转换。

Object类中的方法

equals方法

equals方法用来检测一个对象是否等于另一个对象。

“==”比较的是内存地址，所以含有复合型数据的实例变量再调用equals时应根据需求对equals()进行重写。

注意：对简单数据类型进行封装的Boolean、Byte、Char、Short、Integer、Long、Float、Double以及String，Date，File类把equals方法重写为判断 值 是否相等。

toString方法

toString方法用来返回对象的字符串表示
在直接输出一个对象时，该对象自动调用toString方法，有时这个方法需要根据要求重写。
